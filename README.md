## 📖 About

This repository contains the solution to **Assignment 1** of the Object-Oriented Programming course. The assignment covers fundamental C++ programming concepts including basic I/O, arithmetic operations, data types, control flow (if-else, switch-case), and bitwise operations.

---

## ✅ Questions Covered

| Q# | Topic | Description |
|---|---|---|
| Q1 | Hello World | Print "Hello World" to the console |
| Q2 | Arithmetic Operations | Input two numbers and display sum, difference, product, and quotient |
| Q3 | Data Types | Read and display integer, float, and character values |
| Q4 | Temperature Conversion | Convert Celsius to Fahrenheit |
| Q5 | Escape Sequences | Demonstrate use of `\n`, `\t`, `\b` escape sequences |
| Q6 | Compound Assignment | Use `+=` and `-=` operators |
| Q7 | Swap Without Temp | Swap two numbers using XOR bitwise operator |
| Q8 | Library Fine | Calculate fine based on number of overdue days |
| Q9 | Calculator | Simple calculator using switch-case |
| Q10 | Even or Odd | Check whether a number is even or odd |

## 🧠 Concepts Used

- Basic Input / Output (`cin`, `cout`)

- Data types: `int`, `float`, `char`
- Escape sequences
- Conditional statements (`if`, `else if`, `else`)

## ▶️ How to Run

1. **Clone this repository:**
   ```bash
   git clone https://github.com/parminderrathour/OOPS-assignment-1.git
   cd OOPS-assignment-1
   ```

2. **Compile the file using g++:**
   ```bash
   g++ parminder_ass1.cpp -o output
   ```

3. **Run the output:**
   ```bash
   ./output
   ```

> 💡 You can also use an online compiler like [onlinegdb.com](https://www.onlinegdb.com/) or [replit.com](https://replit.com/) to run C++ code without any setup.
